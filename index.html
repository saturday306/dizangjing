《地藏菩萨本愿经》简介

《地藏菩萨本愿经》，简称《地藏经》，是佛教重要经典之一，主要讲述地藏菩萨的本愿和功德。地藏菩萨以大愿著称，誓愿"地狱不空，誓不成佛；众生度尽，方证菩提"。本经共十三品，详细叙述了地藏菩萨在过去无量劫中，为救度众生所发的宏大誓愿，以及佛陀对弟子们的教诲。阅读此经，有助于增长信心，积累功德，消除业障。

第二品 观察业报

第三品 生死难渡
ěr尔 shí时 huì会 zhōng中 。 yǒu有 pú菩 sà萨 mó摩 hē诃 sà萨 。 míng名 yuē曰 jiān坚 láo牢 。 bái白 fú佛 yán言 。 shì世 zūn尊 。 wǒ我 jīn今 jiàn见 dì地 cáng藏 pú菩 sà萨 。 mò莫 néng能 chēng称 tàn叹 。 ruò若 rén人 yǐ以 sì四 shì事 gōng供 yǎng养 。 yú于 bǎi百 jié劫 zhōng中 。 xiū修 liù六 dù度 xíng行 。 bù不 jí及 dì地 cáng藏 pú菩 sà萨 yī一 niàn念 gōng功 dé德 。 fú佛 gào告 jiān坚 láo牢 pú菩 sà萨 。 rú如 shì是 rú如 shì是 。 rǔ汝 suǒ所 yán言 zhě者 。 fēi非 wèi为 yī一 fú佛 èr二 fú佛 sān三 sì四 wǔ五 fú佛 。 yú于 bǎi百 qiān千 fú佛 suǒ所 。 zhǒng种 zhū诸 shàn善 gēn根 。 dì地 cáng藏 pú菩 sà萨 wēi威 shén神 cí慈 bēi悲 。 bù不 kě可 sī思 yì议 。 shì是 gù故 wǒ我 shuō说 shì是 pú菩 sà萨 。 yǒu有 rú如 shì是 děng等 bù不 kě可 sī思 yì议 dà大 wēi威 shén神 lì力 。 duō多 suǒ所 ráo饶 yì益 。 rǔ汝 dāng当 shēn深 xìn信 。 mò莫 shēng生 yí疑 xīn心 。 jiān坚 láo牢 pú菩 sà萨 bái白 fú佛 yán言 。 shì世 zūn尊 。 wǒ我 dāng当 shǒu守 hù护 dì地 cáng藏 pú菩 sà萨 。 lìng令 zhū诸 zhòng众 shēng生 。 dé得 dà大 lì利 yì益 。 fú佛 yán言 shàn善 zāi哉 。
第四品 地藏本愿
ěr尔 shí时 huì会 zhōng中 。 yǒu有 guǐ鬼 wáng王 míng名 yuē曰 dú毒 guǐ鬼 。 míng名 yuē曰 dàn啖 qì气 。 míng名 yuē曰 dàn啖 jīng精 qì气 。 míng名 yuē曰 dàn啖 xuè血 ròu肉 。 míng名 yuē曰 dàn啖 tāi胎 。 míng名 yuē曰 dàn啖 luǎn卵 。 míng名 yuē曰 dàn啖 rén人 。 míng名 yuē曰 dàn啖 fēi非 rén人 。 míng名 yuē曰 yè夜 chā叉 。 míng名 yuē曰 luó罗 shā刹 。 míng名 yuē曰 pí毗 shě舍 dū阇 。 míng名 yuē曰 fù富 dān单 nà那 。 míng名 yuē曰 jié羯 zhā吒 lán蓝 pó婆 。 míng名 yuē曰 wū乌 mó摩 lēi勒 chā叉 。 míng名 yuē曰 bì薜 lì荔 duō多 。 míng名 yuē曰 jiū鸠 pán盘 tú荼 。 míng名 yuē曰 è饿 guǐ鬼 。 míng名 yuē曰 chù畜 shēng生 。 míng名 yuē曰 dì地 yù狱 。 rú如 shì是 děng等 zhòng众 。 jù俱 lái来 jí集 huì会 。 jiē皆 shì是 dà大 pú菩 sà萨 。 wèi为 huà化 zhòng众 shēng生 。 xiàn现 wèi为 guǐ鬼 wáng王 。 shí时 fú佛 gào告 zhū诸 guǐ鬼 wáng王 jí及 wéi韦 tuó驮 pú菩 sà萨 。 rǔ汝 děng等 dāng当 zhī知 。 dì地 cáng藏 pú菩 sà萨 。 yú于 wú无 liàng量 jié劫 。 fā发 dà大 shì誓 yuàn愿 。 dù度 tuō脱 zhòng众 shēng生 。 rǔ汝 děng等 jiē皆 dāng当 yōng拥 hù护 。 lìng令 qí其 yuán圆 mǎn满 。 shí时 zhū诸 guǐ鬼 wáng王 wén闻 fú佛 suǒ所 shuō说 。 jiē皆 dà大 huān欢 xǐ喜 。 hé合 zhǎng掌 gōng恭 jìng敬 。 yī一 xīn心 tīng听 fǎ法 。
第五品 阎罗众生
ěr尔 shí时 yán阎 luó罗 tiān天 zi子 。 hú胡 má麻 yí仪 děng等 。 qiān千 wàn万 yán阎 luó罗 。 jù俱 cóng从 dì地 chū出 。 zhì至 fú佛 suǒ所 zhù住 。 tóu头 miàn面 lǐ礼 zú足 。 yú于 yī一 miàn面 zuò坐 。 bái白 fú佛 yán言 。 shì世 zūn尊 。 wǒ我 děng等 jīn今 rì日 。 chéng承 fú佛 wēi威 shén神 。 dé得 jiàn见 shì世 zūn尊 。 duō多 shēng生 huān欢 xǐ喜 。 wéi唯 yuàn愿 shì世 zūn尊 。 wèi为 shuō说 yīn因 yuán缘 。 shí时 fú佛 gào告 yán阎 luó罗 tiān天 zi子 。 rǔ汝 shàn善 nán男 zi子 。 rǔ汝 děng等 zài在 guò过 qù去 wú无 liàng量 jié劫 zhōng中 。 jù俱 wèi为 dì地 cáng藏 pú菩 sà萨 。 jiào教 huà化 zhòng众 shēng生 。 jīn今 dé得 chéng成 fú佛 。 huò或 wèi为 tiān天 zi子 。 gè各 lǐng领 yán阎 fú浮 tí提 zhòng众 shēng生 。 shí时 yán阎 luó罗 tiān天 zi子 bái白 fú佛 yán言 。 shì世 zūn尊 。 yán阎 fú浮 tí提 zhòng众 shēng生 。 zào造 è恶 yè业 zhě者 。 shí十 yǒu有 bā八 jiǔ九 。 zào造 shàn善 yè业 zhě者 。 shí十 wú无 yī一 èr二 。 wéi唯 yuàn愿 shì世 zūn尊 。 dà大 bēi悲 cí慈 āi哀 。 wèi为 wǒ我 děng等 shuō说 。 lìng令 zhū诸 zhòng众 shēng生 。 zhī知 suǒ所 qū趋 bì避 。 fú佛 gào告 yán阎 luó罗 tiān天 zi子 。 rǔ汝 shàn善 nán男 zi子 。 ruò若 yǒu有 zhòng众 shēng生 。 bù不 xiào孝 fù父 mǔ母 。 huò或 zhì至 è恶 nì逆 。 zuò作 rú如 cǐ此 zhě者 。 suǒ所 dé得 zuì罪 yè业 。 jiē皆 guī归 dì地 cáng藏 。 dì地 cáng藏 pú菩 sà萨 。 yú于 yán阎 fú浮 tí提 。 yǒu有 dà大 yīn因 yuán缘 。 ruò若 jiàn见 zhòng众 shēng生 。 xiū修 shàn善 zhǒng种 zhǒng种 。 huò或 bù布 shī施 chí持 jiè戒 。 rěn忍 rǔ辱 jīng精 jìn进 。 chán禅 dìng定 zhì智 huì慧 。 jiē皆 dāng当 hù护 niàn念 。 lìng令 dé得 chéng成 jiù就 。
第六品 阎王受记
ěr尔 shí时 shì世 zūn尊 。 cóng从 xiōng胸 yì臆 zhī之 jiān间 。 bǎi百 bǎo宝 mén门 chū出 。 yī一 yī一 bǎo宝 guāng光 。 biàn遍 zhào照 shí十 fāng方 。 zhǒng种 zhǒng种 bǎo宝 tǎ塔 。 mí弥 fù覆 xū虚 kōng空 。 shí时 dì地 cáng藏 pú菩 sà萨 。 cóng从 dì地 yǒng涌 chū出 。 dào到 fú佛 suǒ所 zhù住 。 tóu头 miàn面 lǐ礼 zú足 。 bái白 fú佛 yán言 。 shì世 zūn尊 。 wǒ我 guān观 yán阎 fú浮 tí提 zhòng众 shēng生 。 jǔ举 shǒu手 dòng动 zú足 。 wú无 bù不 shì是 yè业 。 ruò若 wú无 zuì罪 bào报 。 yì亦 kǔ苦 wú无 qióng穷 。 ruò若 yǒu有 zuì罪 bào报 。 yì亦 kǔ苦 wú无 qióng穷 。 wéi唯 yuàn愿 shì世 zūn尊 。 shuō说 dì地 yù狱 míng名 。 jí及 zhū诸 zuì罪 bào报 。 lìng令 hòu后 shì世 zhòng众 shēng生 。 zhī知 suǒ所 jǐng警 jiè戒 。 fú佛 gào告 dì地 cáng藏 pú菩 sà萨 。 shàn善 zāi哉 shàn善 zāi哉 。 rǔ汝 jīn今 wèi为 bēi悲 mǐn悯 yán阎 fú浮 tí提 zhòng众 shēng生 。 jí及 wèi未 lái来 xiàn现 zài在 zhū诸 zhòng众 shēng生 gù故 。 wèn问 wǒ我 dì地 yù狱 zhī之 shì事 。 rǔ汝 jīn今 dì谛 tīng听 。 dāng当 wèi为 rǔ汝 shuō说 。 shí时 dì地 cáng藏 pú菩 sà萨 bái白 fú佛 yán言 。 wéi唯 rán然 shì世 zūn尊 。 yuàn愿 lè乐 yù欲 wén闻 。
第七品 地狱名号
ěr尔 shí时 shì世 zūn尊 。 zàn赞 dì地 cáng藏 pú菩 sà萨 yán言 。 shàn善 nán男 zi子 。 wǒ我 yǐ以 fú佛 yǎn眼 guān观 shì是 yán阎 fú浮 tí提 zhòng众 shēng生 。 jǔ举 dòng动 dòng动 niàn念 。 wú无 bù不 shì是 yè业 。 wú无 bù不 shì是 zuì罪 。 hé何 kuàng况 zì恣 qíng情 shā杀 hài害 。 qiè窃 dào盗 xié邪 yín淫 。 wàng妄 yǔ语 liǎng两 shé舌 。 è恶 kǒu口 qǐ绮 yǔ语 。 tān贪 jí嫉 huì恚 。 chī痴 。 rú如 shì是 zhū诸 zuì罪 。 shuō说 bù不 néng能 jǐn尽 。 shì世 zūn尊 。 yòu又 yán阎 fú浮 tí提 zhòng众 shēng生 。 qí其 xìng性 gāng刚 qiáng强 。 nán难 kě可 diào调 fú伏 。 yú于 shì世 zūn尊 。 shuō说 yī一 jì偈 zàn赞 。 nǎi乃 zhì至 zào造 è恶 。 duò堕 yú于 dì地 yù狱 。 shòu受 zhū诸 kǔ苦 nǎo恼 。 wú无 liàng量 wú无 biān边 。 yī一 jié劫 zhī之 jiān间 。 shù数 bǎi百 wàn万 suì岁 。 fù复 zhuǎn转 tā他 dào道 。 jí即 qí其 zhōng中 dào道 。 duò堕 yú于 dì地 yù狱 。 dì地 yù狱 zhī之 zhōng中 。 shòu受 zuì罪 bù不 yǐ已 。 fù复 zhuǎn转 è饿 guǐ鬼 dào道 。 shòu受 kě渴 yǎng仰 kǔ苦 。 qí其 kǔ苦 wú无 liàng量 。 ruò若 chū出 è饿 guǐ鬼 。 fù复 shòu受 chù畜 shēng生 。 pī披 máo毛 dài戴 jiǎo角 。 xiāng相 hù互 shì噬 dàn噉 。 shēn身 fù负 xíng刑 zhàng杖 。 rì日 yuè月 jié劫 shù数 。 shòu受 jǐn尽 。 zhuǎn转 wèi为 rén人 shēn身 。 jù具 liù六 gēn根 zhě者 。 bǎi百 qiān千 wàn万 fēn分 。 bù不 dé得 yī一 èr二 。
第八品 阎王护法
ěr尔 shí时 yán阎 luó罗 tiān天 zi子 。 bái白 fú佛 yán言 。 shì世 zūn尊 。 wǒ我 jīn今 dāng当 bǎo保 hù护 shì是 yán阎 fú浮 tí提 rén人 。 jí及 zhū诸 guǐ鬼 shén神 。 lìng令 qí其 ān安 yǐn隐 。 bù不 shòu受 kǔ苦 bào报 。 shí时 fú佛 gào告 yán阎 luó罗 tiān天 zi子 。 shàn善 zāi哉 shàn善 zāi哉 。 rǔ汝 jīn今 fā发 xīn心 。 bǎo保 hù护 zhòng众 shēng生 。 shì是 shàn善 nán男 zi子 。 rǔ汝 dāng当 zhī知 zhī之 。 ruò若 yǒu有 zhòng众 shēng生 。 zào造 zhū诸 è恶 yè业 。 dāng当 duò堕 dì地 yù狱 。 rǔ汝 děng等 bù不 yīng应 jiù救 hù护 。 ruò若 zào造 shàn善 yè业 。 yīng应 dé得 kuài快 lè乐 。 rǔ汝 dāng当 jiù救 hù护 。 lìng令 qí其 dé得 ān安 。 shí时 yán阎 luó罗 tiān天 zi子 bái白 fú佛 yán言 。 wéi唯 rán然 shì世 zūn尊 。 dāng当 yī依 fú佛 jiào教 。 hù护 fǎ法 hù护 shēng生 。
第九品 福田利益
ěr尔 shí时 huì会 zhōng中 。 yǒu有 zhū诸 tiān天 guǐ鬼 shén神 。 jù俱 cóng从 zuò座 qǐ起 。 zhěng整 yī衣 fú服 。 hú胡 guì跪 hé合 zhǎng掌 。 bái白 fú佛 yán言 。 shì世 zūn尊 。 wǒ我 děng等 yuàn愿 yú于 wèi未 lái来 shì世 zhōng中 。 bǎo保 hù护 shì是 dì地 cáng藏 pú菩 sà萨 。 jiào教 huà化 zhòng众 shēng生 。 lìng令 qí其 yuǎn远 lí离 è恶 dào道 。 qīn亲 jìn近 fú佛 fǎ法 。 shí时 fú佛 gào告 zhū诸 tiān天 guǐ鬼 shén神 。 rǔ汝 děng等 shàn善 nán男 zi子 。 ruò若 néng能 jiàn见 dì地 cáng藏 pú菩 sà萨 xíng形 xiàng像 。 jí及 wén闻 dì地 cáng藏 pú菩 sà萨 míng名 zì字 。 huò或 zàn赞 tàn叹 。 huò或 gōng供 yǎng养 。 huò或 chēng称 míng名 。 rú如 shì是 děng等 bèi辈 。 yú于 bǎi百 qiān千 shēng生 。 cháng常 dé得 liù六 gēn根 。 jù具 zú足 guāng光 míng明 。 yī衣 shí食 fēng丰 zú足 。 nǎi乃 zhì至 dé得 wén闻 yī一 jù句 yī一 jì偈 。 yú于 bǎi百 qiān千 shēng生 。 cháng常 wèi为 nán男 zi子 。 nǚ女 shēn身 huà化 chéng成 。 shí时 zhū诸 tiān天 guǐ鬼 shén神 wén闻 fú佛 suǒ所 shuō说 。 jiē皆 dà大 huān欢 xǐ喜 。 yǒng踊 yuè跃 wú无 liàng量 。
第十品 利益存亡
ěr尔 shí时 shì世 zūn尊 。 zàn赞 mó摩 yé耶 fū夫 rén人 yán言 。 shàn善 zāi哉 shàn善 zāi哉 。 rǔ汝 jīn今 wèi为 bēi悲 mǐn悯 dāo忉 lì利 tiān天 zhòng众 。 jí及 yán阎 fú浮 tí提 zhū诸 zhòng众 shēng生 gù故 。 wèn问 dì地 cáng藏 pú菩 sà萨 lì利 yì益 zhī之 shì事 。 rǔ汝 dāng当 dì谛 tīng听 。 wǒ我 jīn今 wèi为 rǔ汝 。 fēn分 bié别 yǎn演 shuō说 。 mó摩 yé耶 fū夫 rén人 bái白 fú佛 yán言 。 wéi唯 rán然 shì世 zūn尊 。 yuàn愿 lè乐 yù欲 wén闻 。 fú佛 gào告 mó摩 yé耶 fū夫 rén人 。 rǔ汝 jīn今 dāng当 zhī知 。 ruò若 yǒu有 nán男 rén人 nǚ女 rén人 。 zài在 jiā家 chū出 jiā家 。 xíng行 shàn善 xíng行 è恶 。 zào造 zuì罪 zào造 fú福 。 jiē皆 yóu由 xīn心 dì地 。 ruò若 jiàn见 dì地 cáng藏 pú菩 sà萨 xíng形 xiàng像 。 wén闻 dì地 cáng藏 pú菩 sà萨 míng名 zì字 。 huò或 chēng称 míng名 。 huò或 zàn赞 tàn叹 。 huò或 gōng供 yǎng养 。 rú如 shì是 děng等 shàn善 。 jiē皆 dé得 fú福 bào报 。 ruò若 zào造 è恶 yè业 。 duò堕 yú于 è恶 dào道 。 shòu受 zhū诸 kǔ苦 nǎo恼 。 jiē皆 yóu由 zì自 zào造 。 fēi非 guān关 tā他 rén人 。 mó摩 yé耶 fū夫 rén人 wén闻 fú佛 suǒ所 shuō说 。 huān欢 xǐ喜 xìn信 shòu受 。
第十一品 如来赞叹
ěr尔 shí时 shì世 zūn尊 。 zàn赞 dì地 cáng藏 pú菩 sà萨 yán言 。 shàn善 nán男 zi子 。 rǔ汝 jīn今 wèi为 zhòng众 shēng生 。 fā发 dà大 shì誓 yuàn愿 。 yú于 bǎi百 qiān千 jié劫 。 dù度 tuō脱 yī一 qiè切 kǔ苦 nǎo恼 zhòng众 shēng生 。 wèi未 lái来 zhī之 shì世 。 ruò若 yǒu有 zhòng众 shēng生 。 wén闻 rǔ汝 míng名 zì字 。 jiàn见 rǔ汝 xíng形 xiàng像 。 wén闻 rǔ汝 shuō说 fǎ法 。 rú如 shì是 děng等 bèi辈 。 jiē皆 dé得 jiě解 tuō脱 。 nǎi乃 zhì至 bǎi百 qiān千 shēng生 zhōng中 。 cháng常 bù不 duò堕 luò落 è恶 dào道 。 shàn善 nán男 zi子 。 rǔ汝 yǐ以 dà大 bēi悲 。 hù护 chí持 shì是 yán阎 fú浮 tí提 rén人 。 yú于 xiàn现 zài在 wèi未 lái来 。 shì世 shì世 cháng常 dé得 ān安 lè乐 。 wǒ我 jīn今 shuō说 jì偈 。 wèi为 rǔ汝 zàn赞 tàn叹 。 dì地 cáng藏 dì地 cáng藏 。 shǐ使 mìng命 fù负 hé荷 。 qīng轻 zhòng重 jiē皆 shòu受 。 jì济 dù度 kǔ苦 qù趣 。 niè涅 pán槃 jì寂 jìng静 。 yǒng永 duàn断 shēng生 sǐ死 。 rú如 shì是 gōng功 dé德 。 bù不 kě可 sī思 yì议 。
第十二品 见闻利益
ěr尔 shí时 fú佛 gào告 sì四 zhòng众 。 dì谛 tīng听 dì谛 tīng听 。 shàn善 sī思 niàn念 zhī之 。 wǒ我 wèi为 rǔ汝 děng等 。 shuō说 dì地 cáng藏 pú菩 sà萨 。 jiàn见 wén闻 lì利 yì益 zhī之 shì事 。 ruò若 yǒu有 zhòng众 shēng生 。 wén闻 dì地 cáng藏 pú菩 sà萨 míng名 zì字 。 huò或 zhān瞻 lǐ礼 xíng形 xiàng像 。 huò或 chēng称 míng名 hào号 。 huò或 zuò作 xiāng香 huǒ火 。 huò或 tú涂 xiān鲜 huā花 。 huò或 shè设 gōng供 jù具 。 rú如 shì是 děng等 rén人 。 yú于 xiàn现 zài在 wèi未 lái来 。 cháng常 dé得 bǎi百 qiān千 guǐ鬼 shén神 。 rì日 yè夜 wèi卫 hù护 。 bù不 wèi为 è恶 shì事 。 bō拨 qí其 ěr耳 lún轮 。 ruò若 rén人 mèng梦 mèi寐 。 jiàn见 guǐ鬼 shén神 děng等 。 jiē皆 dāng当 gōng恭 jìng敬 。 shì是 dì地 cáng藏 pú菩 sà萨 。 huà化 xiàn现 qí其 shēn身 。 lìng令 qí其 jiě解 tuō脱 。
第十三品 校量布施
ěr尔 shí时 shì世 zūn尊 。 zàn赞 dì地 cáng藏 pú菩 sà萨 yán言 。 shàn善 nán男 zi子 。 rǔ汝 yú于 guò过 qù去 wú无 liàng量 jié劫 。 fā发 dà大 shì誓 yuàn愿 。 dù度 tuō脱 zhòng众 shēng生 。 qí其 yuàn愿 shēn深 zhòng重 。 bù不 kě可 sī思 yì议 。 ruò若 yǒu有 zhòng众 shēng生 。 yǐ以 qī七 bǎo宝 mǎn满 sān三 qiān千 dà大 qiān千 shì世 jiè界 。 gōng供 yǎng养 yú于 fú佛 。 jí及 zhū诸 xián贤 shèng圣 。 rú如 shì是 bù布 shī施 。 gōng功 dé德 wú无 liàng量 。 ruò若 fù复 yǒu有 rén人 。 jiàn见 dì地 cáng藏 pú菩 sà萨 xíng形 xiàng像 。 nǎi乃 zhì至 yī一 chēng称 zàn赞 。 yī一 lǐ礼 bài拜 。 shì是 rén人 bǎi百 fǎn返 。 sān三 shí十 èr二 wáng王 zhōng中 。 wèi为 shēng生 dì帝 wáng王 。 qiān千 fǎn返 。 shēng生 sān三 shí十 sān三 tiān天 。 cháng常 bù不 wèi为 bēi卑 jiàn贱 。 nǎi乃 zhì至 bǎi百 qiān千 shēng生 。 zhōng中 。 cháng常 dé得 wén闻 xìng姓 míng名 。 xiāng相 mào貌 duān端 zhèng正 。 shòu受 rén人 gōng恭 jìng敬 。 shì是 gù故 rǔ汝 dāng当 xuān宣 yáng扬 。 lìng令 hòu后 shì世 zhòng众 shēng生 。 zhī知 cǐ此 gōng功 dé德 。 dì地 cáng藏 pú菩 sà萨 yán言 。 wéi唯 rán然 shì世 zūn尊 。 yuàn愿 wèi为 zhòng众 shēng生 。 fēn分 bié别 yǎn演 shuō说 。 shí时 fú佛 huān欢 xǐ喜 。 chì敕 wén文 shū殊 shī师 lì利 。 lìng令 guǎng广 wèi为 dà大 zhòng众 。 jiě解 shuō说 dì地 cáng藏 pú菩 sà萨 。 běn本 yuàn愿 gōng功 dé德 。
